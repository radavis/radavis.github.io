## sqlite3 command line

```
$ sqlite3 database-name.db
sqlite> .help
sqlite> .tables
sqlite> .schema
sqlite> .headers on
sqlite> .mode column
sqlite> select * from table_name limit 40;
sqlite> .q
```

## `~/.sqliterc`

```
.headers on
.mode column
```

## sqlite3 datatypes

* null
* integer
* real
* text
* blob
