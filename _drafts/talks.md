# Talks


**Authentication and Authorization**

- Goal: Understand the basics of Auth, and how Web Auth systems work.
- Topics:
  * OAuth
  * JWT
  * Casport, X.509 Certificates
  * Application of these Auth schemes to HTTP


**Docker Basics** (Docker Applied to Web Technologies)

- Goal: Introduce Docker, and the types of problems that are solved with
"containerization."
- Topics:
  * Installation
  * Docker commands
  * Example `Dockerfile` and usage
  * Steps: create an image, use an image (container), deploy
  * Port management
  * Full-blown webapp with docker-compose


**Creating CI/CD pipelines with GitLab**

- Goal: Understand what GitLab does (everything), and how we can leverage this
service to build a CI/CD pipeline.
- Topics:
  * Compare GitHub and GitLab
  * GitLab features
  * What the heck is a pipeline, anyway?
  * Example: code, test, build, deploy, monitor


**Code Review: A HowTo Guide**

- Goal: Convey the importance of regular code review, and how to integrate
this strategy into day-to-day operations.
- Topics:
  * Why should we do code review?
    - Fewer bugs
    - Knowledge sharing
    - Build team rapport
  * Effective strategies for good code review
  * Example Steps: read the ticket, pull the code, run the test suite,
  click-test, read the code, leave positive feedback.
