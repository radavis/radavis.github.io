# Screenly

>
Screenly OSE is the most popular open source digital signage software for the
Raspberry Pi. Use it for free (it's a freeware after all) and save money.

Follow instructions for [Pi Config](/pi-config).

```
$ bash <(curl -sL https://www.screenly.io/install-ose.sh)
```

[homepage](https://www.screenly.io/ose/)
