## Key Values

[source](http://keyvalues.io)

These are the key values we strive to achieve. Some of these key values
we excel at. Others, we need to improve upon. The idea being that we are always
thinking about these values, and adjusting our behavior to align with them.

### Daily Routines

* Thoughtful Office Layout
* Having Breakfast/Lunch Together

### Strategy

* Product-Driven
* Customer Comes First

### Career Growth

* Good for Developers of all experience levels
* Continuous Feedback is provided, if you want it

### Personal Health

* Work/Life Balance
* Safe Environment to Fail
* Fosters Psychological Safety

### Team Values

* Cross-Department Collaboration
* Open Communication
* EQ > IQ

### Engineering

* Continuous Delivery
* Agile Methods
  - User Stories, Acceptance Criteria
  - morning standup meeting
* Pair Programming
  - on code reviews
  - when we're stuck on a problem
* Start-to-Finish Ownership
  - taking on a task to completion
  - asking for help when you need it
