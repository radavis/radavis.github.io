# Java Koans

Work through the Java Koans project at your own pace. To get started, type out 
the following commands in your terminal.

```bash
$ curl -L -o java-koans.zip https://github.com/matyb/java-koans/archive/master.zip 
$ unzip java-koans.zip
$ rm java-koans.zip
$ cd java-koans-master/koans
$ ./run.sh
```
