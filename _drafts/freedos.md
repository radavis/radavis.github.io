# FreeDOS

Try out GalaxyWrite for DOS. It's a shareware clone of WordPerfect.

My Introduction to Unix/Linux class was boring. Same goes for my OS class.
Create a course that explains all of the parts of an OS, in a beginner-friendly
way.

  * labs for trying out commands
  * explain concepts in plain English
  * open source, allow for translations (built-in internationalization)
  * pay for print copy
  * intentional ordering of topics **with applications of those topics as exercises/labs**

FreeDOS Uses

  * play classic DOS games
  * run legacy software
  * develop embedded systems

_23 Years of FreeDOS_ is kind of a hard read for someone who isn't directly
involved in the project.
