# Web Design

### Content

> The purpose of design is to enhance the presentation of the content it is
> applied to.

> Written content makes up for more than 90% of the Web.

### Centering

Limit the number of characters per line.

### Font-family

Choose Primary and Secondary fonts, using fallback fonts.

### Spacing

Provide space both around and within the content.

### Color and Contrast

Use a softer shade of black for text.

Choose a primary color to add accents on interactive elements, such as links.

Choose secondary colors to be used on borders and backgrounds.

### Images

> Graphics and icons can be used either as ornaments to support your content,
> or actively take part in the message you want to convey.

Choose or create a logo.
