Solutions for Power Monitoring

Idea: small networked linux pc with power monitoring capabilities

* Determine GPIO Voltage Requirements (0-5VDC, 0-3VDC)
* Measure Voltage/Current 120V-220VAC, 15A
* Determine Accuracy of Measurements
* Software Interface:
  - Dumb API
  - Push Notifications
  - Publish/Subscribe
* Software Configuration
  - Configuration for home networking devices is the absolute worst.
  - Simple web-enabled configuration dashboard (wifi settings, post url, post frequency)
* Hardware
  - Ethernet
  - 802.11
  - Single red notification LED (flash on power up, solid when network connected)
  - Reset switch
  - Plug-outlet packaging, Wall-wart packaging
