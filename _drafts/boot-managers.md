# Boot Managers

## For BIOS PCs (pre 2007)

* GAG Graphical Boot Manager [homepage](http://gag.sourceforge.net/index.html)
* grub4dos [homepage](http://grub4dos.sourceforge.net/)

## For EFI & UEFI PCs (post 2007)

* rEFInd [homepage](http://www.rodsbooks.com/refind/)
* burg [homepage](https://code.google.com/archive/p/burg/)
* GRUB2 [homepage](https://wiki.gentoo.org/wiki/GRUB2)
