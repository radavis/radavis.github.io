# Web Application Walkthroughs

Amazon's tutorial for getting started with Docker containers steps you through
the process of using their Elastic Container Service (ECS) using instructions
steps with screenshots.

[source](https://aws.amazon.com/getting-started/hands-on/deploy-docker-containers/)

How could someone replicate this system?

* Capybara/Cypress/Protractor tests to interact with the page, assert
  functionality, take screenshots.
* Method for documenting a step-by-step process. Custom markdown -> web form.
* Pair together, interleave steps and tests.
