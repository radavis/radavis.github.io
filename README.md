# radavis.github.io

My blog.

## Getting Started

```bash
$ git clone git@github.com:radavis/radavis.github.io blog
$ cd blog
$ bundle
$ alias be="bundle exec"
$ be jekyll serve
```

Visit http://localhost:4000
