# Video and Media Cataloging App

Name: LaserDisc Catalog

## Features

* [ ] OAuth w/ Facebook & Google
* [ ] Add movie to collection (via Title/Format)
* [ ] Add movie to collection (via ISBN, Barcode)
* [ ] View collection
* [ ] Add friend
* [ ] View friend's collection
* [ ] Request to borrow

## Notes

```
$ rails new laserdisc-catalog -m https://raw.github.com/RailsApps/rails-composer/master/composer.rb
```
