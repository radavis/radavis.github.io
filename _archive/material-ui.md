## Box Component

```jsx
<Box component="span">
  This box will render as a span.
</Box>
```

[box docs](https://material-ui.com/components/box/)
[box api](https://material-ui.com/system/api/)

## Spacing Properties

```jsx
<Box m={1} p={1}>Add some padding</Box>
```

[spacing api](https://material-ui.com/system/spacing/#api)


## Color Properties

```jsx
<Box color="primary.main"></Box>
```

[palette api](https://material-ui.com/system/palette/#api)


## Border Properties

```jsx
<Box border={1}>
  This box has a border of one spacing unit
</Box>
```
